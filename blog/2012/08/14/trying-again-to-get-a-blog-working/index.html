Trying Again to Get a Blog Working

Aug 14th, 2012

I have tried to get a static blog/website with nanoc earlier, but I have found that I’m not really up to making it too well, so I was looking a bit over on jekyll and octopress, and this seems like something that I really can use.

So there is one thing that I think that I have learned from this experience, that is, for getting something done, go with the simplest thing that can work. For learning just go into something, even though you have to go out on a bit thin ice, the result may not be that stellar, but for sure you’ll learn quite a bit in the process.
